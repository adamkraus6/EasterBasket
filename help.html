4 Tier: Extensions 30

Extension 1: 1a 5pts CSS feature

I used CSS flexbox for displaying the basket and goodies. The basket gets the whole width and each goody gets 1/6 of the width. Flexbox easily allows me to set how items wrapped, which is done left to right, then upwards. I knew about flexbox before, and used CSS Flexbox Guide to help. The basket and goodies are all contained within the same flexbox.

Extension 2: 1b 5pts Fancy background

I made the page background a gradient using CSS, which I used CSS Gradient to design. It fades from #b2bec3 to #81ecec at the bottom of the page.

Extension 3: 2d 10pts Drag and drop uploading

You can drag and drop files to upload them on the file management page. You can drag a file into the noted area and it will automatically be uploaded.

Extension 4: 2i 5pts Standard form selections

I added a combo box to select which goody is being added, and then an add egg button to add the selected goody.

Extension 5: 3a 5pts 3 more goodies

I added 2 goodies, pale and dirt eggs, and a grey basket option.
